# kable
Kable gets its name from my own name plus the suffix "-able".
## How can it do
It supports all features that other programming languages have.
Currently, it is still under development.
## Why I created Kable
Kable is an interpreted programming language like *Java*.
I made it and avoid receiving legal notices from Oracle over Java-related issues.
Kable has numerous advantages here.
1. High Performance
Kable will be extremely fast and much faster than *Python*. Maybe PSF want to issue a warrant for someone's arrest
2. Programming modernization
It follows modern design ideas. There are no complicated and intricate syntax, rigid keywords, nor tedious expressions to trouble developers.
3. Too bright
Kable is tiny and so bright. It is fully written in C++ and you can see all of it.
4. Cross-platform
Powered by its built-in virtual machine, it can run programs smoothly across different platforms.
